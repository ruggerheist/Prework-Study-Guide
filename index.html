HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements define a section in the document containing thematic content
  • The img element contains information about the images displayed on the webpage
  • The br element creates a line of empyy space or line break between 2 blocks of content
  • The footer element contains author, copywrite, sitemap, navigation
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The meta element defines metadata i.e. character set, description, keywords, author, and viewport
  • The script element embedws client side scripts in the HTML doc

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A link element creates the connection to the external file
  • Declarations contain the CSS property that the value is applied to
  • DO NOT REPEAT YOURSELF (DRY) encourages reuse of code to prevent redundancy

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "message"
  • git pull origin main to pull down latest main branch
