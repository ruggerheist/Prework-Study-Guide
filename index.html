HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements define a section in the document containing thematic content
  • The img element contains information about the images displayed on the webpage
  • The br element creates a line of empyy space or line break between 2 blocks of content
  • The footer element contains author, copywrite, sitemap, navigation
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The meta element defines metadata i.e. character set, description, keywords, author, and viewport
  • The script element embedws client side scripts in the HTML doc
